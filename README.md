# 🔒 AgentGuard: Защита от EscalationVia LLM

**Дипломный проект** на тему: *Защита системных агентов и интеграционных сервисов от эскалации привилегий, вызванной атаками на LLM*

## 🎯 О проекте

AgentGuard - это система защиты, предназначенная для предотвращения эскалации привилегий системных агентов и интеграционных сервисов через атаки на Language Models (LLM). Проект решает актуальную проблему безопасности при использовании LLM в системных решениях.

## 📋 Цель работы

Разработать и экспериментально оценить механизм предотвращения эскалации привилегий системных агентов и интеграционных сервисов, вызванной атаками на LLM, с реализацией проекта на Python.

## ✅ Основные задачи

- ✓ Провести системный анализ угроз безопасности (OWASP Top 10 for LLM Applications)
- ✓ Проанализировать архитектуры системных агентов и уязвимые точки
- ✓ Исследовать типы атак на LLM, опасные с точки зрения эскалации привилегий
- ✓ Разработать комбинированный метод обнаружения и предотвращения атак
- ✓ Реализовать MVP механизма защиты на Python
- ✓ Провести экспериментальную оценку эффективности метода

## 🏗️ Основные компоненты системы

### 1. **Семантический анализ**
   - Анализ намерений пользовательских команд
   - Обнаружение скрытых инструкций (prompt injection)
   - Выявление попыток манипуляции контекстом

### 2. **Оценка рисков команд**
   - Анализ требуемых привилегий для выполнения команд
   - Проверка соответствия между запрашиваемыми и доступными привилегиями
   - Оценка потенциального ущерба от выполнения команды

### 3. **RAG (Retrieval-Augmented Generation)**
   - Поиск релевантных примеров безопасных операций
   - Контекстуализация решений на основе примеров из базы знаний
   - Улучшение точности оценки угроз

### 4. **Агентский граф (LangGraph)**
   - Оркестрация процесса анализа
   - Координация компонентов защиты
   - Логирование и аудит всех операций

## 🚀 Быстрый старт

### 1️⃣ Требования

- Python 3.8+
- Ollama с моделью llama2
- Не менее 4GB ОЗУ

### 2️⃣ Установка

```bash
# Клонируем репозиторий
git clone https://github.com/lalmark/agentguard-llm-security.git
cd SecurityAIAgent

# Создаём виртуальное окружение
python3 -m venv venv

# Активируем
source venv/bin/activate  # Linux/macOS
# или
venv\Scripts\activate  # Windows

# Устанавливаем зависимости
pip install -r requirements.txt
```

### 3️⃣ Конфигурация

Создайте файл `.env`:

```env
MODEL_NAME=llama2
```

### 4️⃣ Запуск Ollama

```bash
# В отдельном терминале
ollama serve
```

Убедитесь, что модель загружена:
```bash
ollama pull llama2
```

### 5️⃣ Запуск приложения

```bash
python -m app.main
```

## 📦 Зависимости

- **langchain** - фреймворк для работы с LLM приложениями
- **langchain-ollama** - интеграция с локальными моделями Ollama
- **langchain-core** - базовые компоненты LangChain
- **langgraph** - библиотека для построения агентских графов
- **python-dotenv** - управление переменными окружения

## 🔍 Как это работает

```python

# Инициализируем защитный агент
security_agent = build_graph()

# Подаём команду/вопрос на анализ
```
